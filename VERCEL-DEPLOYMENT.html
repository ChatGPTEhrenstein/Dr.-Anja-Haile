📋 Schritt-für-Schritt Anleitung:

Schritt 1: Vercel Account erstellen

  1. Gehen Sie zu vercel.com
  2. Klicken Sie auf "Sign Up"
  3. Wählen Sie "Continue with GitHub" oder "Continue with Google"
  4. Kostenloses Hobby-Konto auswählen
✅ Account kostenlos - keine Kreditkarte erforderlich!

Schritt 2: Projekt erstellen

  1. Nach dem Login: Klicken Sie auf "New Project"
  2. Wählen Sie "Browse All Templates"
  3. Oder direkt: "Deploy" Button
📸 Dashboard → New Project → Import bestehende Datei

Schritt 3: Dateien hochladen

Option A - Drag & Drop (Empfohlen):

  1. Die Datei dr-anja-haile-vercel.tar.gz herunterladen
  2. Datei entpacken (zip/tar extrahieren)
  3. Alle Dateien auswählen und per Drag & Drop auf Vercel ziehen

Option B - GitHub Repository:

  1. GitHub Repository erstellen
  2. Dateien hochladen
  3. Vercel mit GitHub verbinden
  4. Repository importieren

Schritt 4: Deployment konfigurieren

Project Name: dr-anja-haile-psychotherapie Framework Preset: Other (Static) Root Directory: ./ Build Command: (leer lassen) Output Directory: (leer lassen) Install Command: (leer lassen)

Schritt 5: Deploy!

  1. Klicken Sie auf "Deploy"
  2. Warten Sie 30-60 Sekunden
  3. 🎉 Ihre URL wird angezeigt!
⚡ Deployment-Zeit: 30-60 Sekunden

🎨 Custom Domain hinzufügen (optional):

  1. Im Vercel Dashboard: Settings → Domains
  2. Neue Domain eingeben: anjahaile.de
  3. DNS-Einstellungen bei Ihrem Domain-Provider ändern:
A Record: @ → 76.76.19.61 CNAME: www → dr-anja-haile-psychotherapie.vercel.app
