🚀 Schritt-für-Schritt Anleitung

1

Netlify Account erstellen

  1. Gehen Sie zu netlify.com
  2. Klicken Sie auf "Sign up"
  3. Wählen Sie "Email" oder "GitHub/Google"
  4. Kostenloses Konto erstellen
✅ Keine Kreditkarte erforderlich!
2

Website deployen (Drag & Drop)

  1. Nach dem Login sehen Sie das Netlify Dashboard
  2. Suchen Sie den Bereich "Want to deploy a new site without connecting to Git?"
  3. Laden Sie dr-anja-haile-netlify.zip herunter und entpacken Sie die Datei
  4. Ziehen Sie den gesamten entpackten Ordner in den Drag & Drop Bereich
📸 Screenshot: Großer Drag & Drop Bereich im Netlify Dashboard
"Drag and drop your site output folder here"
🎉 Automatisches Deployment startet sofort!
3

Deployment abwarten

Netlify verarbeitet Ihre Dateien automatisch:

  • 📤 Upload der Dateien (10-20 Sekunden)
  • ⚙️ Verarbeitung und Optimierung (30-60 Sekunden)
  • 🌐 Bereitstellung auf globalem CDN (30 Sekunden)
Status-Anzeige: Sie sehen den Fortschritt live im Dashboard
4

Ihre Website ist live!

Nach erfolgreichem Deployment erhalten Sie:

URL: https://[zufallsname].netlify.app Beispiel: https://amazing-cupcake-123abc.netlify.app
✅ Ihre Dr. Anja Haile Website ist jetzt online!

🎨 Website-Name personalisieren (Optional)

5

Schöne URL erstellen

  1. Klicken Sie im Dashboard auf "Site settings"
  2. Gehen Sie zu "General" → "Site details"
  3. Klicken Sie auf "Change site name"
  4. Wählen Sie einen Namen, z.B.:
Vorschläge: • dr-anja-haile • anja-haile-psychotherapie • psychotherapie-anja-haile • praxis-dr-haile

Neue URL: https://dr-anja-haile.netlify.app

🌐 Custom Domain hinzufügen (Optional)

6

Eigene Domain verwenden

Falls Sie eine eigene Domain haben (z.B. anjahaile.de):

  1. Im Dashboard: "Domain settings"
  2. Klicken Sie "Add custom domain"
  3. Domain eingeben: www.anjahaile.de
  4. DNS-Einstellungen bei Ihrem Domain-Provider ändern:
DNS-Einstellungen: CNAME: www → [ihr-site-name].netlify.app A Record: @ → 75.2.60.5
🔒 SSL-Zertifikat wird automatisch erstellt!
